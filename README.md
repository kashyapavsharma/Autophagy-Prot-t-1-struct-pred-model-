# Autophagy-Prot-t-1-struct-pred-model-
takes pdb matrices and converts the same into t+1 stage, and outputs in pdb matrix format, uses the xyz columns andthe correlation column from the pdb matrix structure, such that we can easily process the data and feed it to the model. Here the model to be used is either an autoencoder and a transformer OR a variational auto encoder.
